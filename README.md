AI Operations Portfolio: Sales Agentic Workflows
I build and operate agentic systems that turn raw prospect data into revenue. I don't just "use" AI tools; I orchestrate them into governed, repeatable workflows.

🚀 The Agent Workflow Story
The Challenge: I handed an end-to-end sales-meeting-prep workflow to AI agents. It initially failed due to context drift, hallucinated facts, and weak next steps.
The Fix: I implemented:

Source-first inputs: Strict validation of prospect data before agent processing.

Strict output schema: Enforced JSON structures to prevent formatting drift.

QA Verdict Gate: A human-in-the-loop review step that compares agent output against a rubric before sending it to the sales team.

Defect Log: A versioned log to track agent failures and iterate on prompt/logic robustness.

🛠 Core Tech Stack
Orchestration: n8n, Python

LLMs: Claude, ChatGPT (prompt engineering & rubrics)

Ops: GitHub (PRs/Branching), Notion (Decision logs/SOPs), HubSpot

Methodology: QA verdict discipline, human-in-the-loop gates, structured deployment handoffs.

📈 Impact
87+ clients closed as a high-ticket sales operator.

~EUR190K revenue generated through structured discovery & AI-assisted follow-up systems.

35% closing rate maintained by standardizing objection libraries and call action plans.
